คู่มือการใช้งาน Kanbann

ระบบจดการบ้าน ทวงงาน แจกสรุปสอบ ประเมินหัวหน้า เลือกตั้ง และจัดการชีวิตประจำวันของห้อง ม.2/3

1. หน้าแรก (Home Page)

เมื่อเข้ามายังเว็บไซต์ คุณจะพบกับหน้าจอหลักที่แบ่งออกเป็น 3 คอลัมน์:

  1. "หน้าสำหรับคนจดงาน" (ซ้าย): สำหรับเข้าไปเพิ่มงานใหม่ ต้องล็อกอินก่อน
  2. "พริมง่วงทวงบุญคุณ" (กลาง): เข้าสู่หน้า กระดานงาน (Kanban Board) เพื่อเช็คงานทั้งหมด
  3. "แจกสรุปสอบกลางภาค" (กลางล่าง): พร้อมนาฬิกานับถอยหลังวันสอบ คลิกเพื่อเข้าดู/แชร์สรุปสอบ
  4. "ผลการเลือกตั้งปี 2569" (ขวา): เข้าดูผลโหวตหัวหน้าห้อง
  5. "ระบบประเมินหัวหน้า" (ขวาล่าง): เข้าไปประเมินหัวหน้าห้องแบบไม่ระบุตัวตน

ปุ่มลัดด้านล่าง

  • "จัดการข้อความ LINE": สั่งส่งข้อความหรือสร้างโพลเข้ากลุ่ม LINE
  • "จัดการผู้สมัคร (Admin)": แก้ไขรายชื่อผู้สมัครเลือกตั้ง
  • "ตั้งค่าระบบพริมจ๋า (Admin)": เปิด-ปิดระบบต่างๆ และส่งออกข้อมูล
  • "ระบบทวงเงินห้อง": จัดการค่าใช้จ่ายของห้อง
หมายเหตุ: ระบบใดที่ถูกปิดจากหน้าตั้งค่า การ์ดจะแสดงป้าย "ปิดระบบชั่วคราว" พร้อมเบลอพื้นหลัง ไม่สามารถกดเข้าได้

2. หน้ากระดานเช็คงาน (Kanban Board)

พื้นที่หลักสำหรับตรวจสอบงานค้างและติดตามความคืบหน้า (เข้าถึงที่ /kanban)

หมายเหตุ: การปรับสถานะงาน (เช่น ติ๊ก "เสร็จแล้ว") หรือลบงานออกจากกระดาน จะมีผลเฉพาะในหน้าจอเครื่องของคุณเท่านั้น ไม่กระทบกับผู้ใช้อื่น

รูปแบบการแสดงผล

  • มุมมองกระดาน (Board View): แบ่งคอลัมน์ (ต้องทำ, กำลังทำ, เสร็จแล้ว) รองรับการลากและวาง (Drag & Drop)
  • มุมมองติ๊ก (List View): แสดงเป็นรายการยาว สามารถกดติ๊กถูกด้านหน้าเพื่องานเปลี่ยนสถานะเป็นเสร็จสิ้น
  • มุมมองหมวดหมู่ (Category View): ระบบจัดกลุ่มให้อัตโนมัติ เช่น งานเลยกำหนด, งานส่งภายใน 7 วัน, และแยกตามรายวิชา

3. การเพิ่มงาน & การแก้ไข

สามารถบันทึกงานใหม่ได้ที่ /add โดยข้อมูลที่จำเป็นต้องกรอก (บังคับ) ได้แก่ ชื่อวิชา, กำหนดส่ง และรายละเอียดงาน

  • ข้อมูลเสริมที่ระบุได้: ครูผู้สั่ง, วิธีการส่งงาน, และการแนบไฟล์รูปภาพ
  • การแก้ไขงาน: ไปที่หน้ากระดาน คลิกเปิดงานที่ต้องการ แล้วเลือก "แก้ไข" (จำเป็นต้องเข้าสู่ระบบก่อน)

4. คลังสรุปสอบ (Summaries)

ระบบแชร์สรุปเนื้อหาสอบกลางภาค อ่านฟรี โหลดฟรี (เข้าถึงที่ /summaries)

การดูสรุปสอบ

  • ค้นหาและกรอง: ค้นหาตามชื่อ, วิชา, หรือชื่อผู้สรุป และกรองตามรายวิชาผ่าน dropdown
  • ดูรูปภาพ (Lightbox): คลิกรูปเพื่อเปิดดูแบบเต็มจอ พร้อมปุ่ม ◀ ▶ เลื่อนดูรูปถัดไป/ก่อนหน้า (หรือกดลูกศรที่คีย์บอร์ดก็ได้)
  • บันทึกรูป: กดปุ่ม "บันทึกรูป" ใน Lightbox เพื่อดาวน์โหลดลงเครื่อง
  • เปิดลิงก์: สรุปที่แนบลิงก์จะมีปุ่ม "เปิดลิงก์" ให้กดเพื่อเปิดใน tab ใหม่
  • แสดง "By": แต่ละการ์ดจะแสดงชื่อผู้สรุป เช่น "By พริม" เป็น tag สีม่วง

การอัปโหลดสรุปสอบ

คลิกปุ่ม "แชร์สรุปสอบ" หรือเข้าที่ /summaries/upload (ต้องล็อกอินก่อน)

  • เลือกประเภท: กดเลือก "แนบไฟล์" (อัปโหลด PDF/รูป) หรือ "แนบลิงก์" (Google Drive, OneDrive ฯลฯ)
  • ข้อจำกัดลิงก์: ลิงก์จาก Instagram ไม่รองรับ — ระบบจะแจ้งเตือนสีแดงทันที
  • เลือกวิชา: เลือกจาก dropdown (คณิตศาสตร์, วิทยาศาสตร์, ภาษาไทย, ภาษาอังกฤษ, สังคมศึกษา, ประวัติศาสตร์, ศิลปะ, สุขศึกษา, การงานอาชีพ, เทคโนโลยี) หรือเลือก "อื่นๆ" แล้วพิมพ์เอง
  • ช่อง "By": ระบุชื่อผู้สรุปเพื่อให้เครดิตเจ้าของผลงาน
หมวดหมู่วิชา: ระบบจัดกลุ่มคำย่ออัตโนมัติ เช่น "วิทย์" = "วิทยาศาสตร์", "คณิต" = "คณิตศาสตร์", "อังกฤษ" = "ภาษาอังกฤษ" — ทำให้ filter ทำงานถูกต้องแม้ตั้งชื่อวิชาต่างกัน

5. ผลการเลือกตั้ง (Election)

แสดงผลโหวตหัวหน้าห้อง ม.2/3 ประจำปี 2569 (เข้าถึงที่ /election)

  • แสดงผู้สมัคร: รูปโปรไฟล์, ชื่อ, หมายเลข และจำนวนคะแนนโหวต
  • จัดการผู้สมัคร (Admin): เพิ่ม แก้ไข ลบ ผู้สมัครได้ที่ /election/edit

6. ระบบประเมินหัวหน้า (Evaluate Boss)

ระบบประเมินผลงานหัวหน้าห้องแบบไม่ระบุตัวตน ธีมสีชมพู (เข้าถึงที่ /evaluate-boss)

  • ให้คะแนนดาว (สีเหลือง): ประเมิน "หน้าตา" และ "ความรับผิดชอบ" จาก 1-5 ดาว
  • เลือกลักษณะนิสัย: กดเลือกได้หลายข้อ (ใจดี, จริงจัง, เฮฮา, ดุ, เท่ ฯลฯ)
  • หัวข้อให้คะแนนเพิ่มเติม: ภาวะผู้นำ, การสื่อสาร, การแก้ปัญหา (แบบ 1-5 ดาว)
  • ข้อเสนอแนะ: พิมพ์ข้อความอิสระ
  • ควรเปลี่ยนหัวหน้าไหม?: ปุ่ม Yes (สีเขียวนีออน) หรือ No (สีแดงแป๊ด)
ความเป็นส่วนตัว: ผลการประเมินจะไม่ผูกกับตัวตนผู้ตอบ ส่งได้แบบ Anonymous

7. การจัดการข้อความ LINE

ระบบแจ้งเตือนและสั่งการบอท LINE เข้ากลุ่มห้อง (เข้าถึงที่ /line)

  • สร้างโพล (Custom Poll): ตั้งคำถามและเพิ่มตัวเลือกได้สูงสุด 10 ข้อ กำหนดเวลาปิดโหวตได้อิสระ
  • ส่งข้อความด่วน: ใช้รูปแบบข้อความสำเร็จรูปที่ตั้งค่าไว้แล้วเพื่อความรวดเร็ว
  • ส่งข้อความกำหนดเอง: พิมพ์ประกาศใดๆ และให้ระบบส่งเข้ากลุ่ม

8. ตั้งค่าระบบ (Settings)

หน้ารวมเครื่องมือจัดการระบบทั้งหมด (เข้าถึงที่ /settings)

เมนูหลัก

  • ตารางสอน: จัดการตารางเรียนแต่ละคาบ
  • ตารางเวร: จัดการเวรทำความสะอาด จันทร์-ศุกร์
  • ชุดเครื่องแบบ: กำหนดชุดนักเรียนที่ต้องใส่ในแต่ละวัน
  • ระบบทั่วไป: เปิด-ปิดระบบต่างๆ ผ่านสวิตช์ toggle

ระบบทั่วไป (/settings/system)

  • เปิด-ปิดระบบ: ระบบจดงาน, กระดานงาน, แจกสรุปสอบ, ผลเลือกตั้ง, ประเมินหัวหน้า — สามารถปิดแต่ละระบบได้ด้วยสวิตช์
  • ส่งออกผลประเมินหัวหน้า: กดปุ่ม "Export CSV" (สีเขียว) เพื่อดาวน์โหลดข้อมูลการประเมินทั้งหมดเป็นไฟล์ CSV เปิดใน Excel ได้ หรือกดปุ่ม "Export PDF" (สีชมพู) เพื่อดาวน์โหลดเป็นเอกสาร PDF พร้อมตารางหัวสีชมพู

9. ระบบทวงเงินห้อง (Funds)

ระบบจัดการค่าใช้จ่ายของห้อง ติดตามว่าใครจ่ายแล้ว ใครยังค้าง (เข้าถึงที่ /funds)

10. หน้า 404 (ไม่พบหน้า)

กรณีเข้า URL ที่ไม่มีอยู่ในระบบ ระบบจะแสดงหน้า "ไม่พบหน้าที่คุณต้องการ" พร้อมเมนูลัดไปยังทุกหน้าหลักของระบบ และปุ่ม "กลับสู่หน้าหลัก"

11. ข้อมูลทางเทคนิค

  • Frontend: Next.js (React) + Tailwind CSS + Framer Motion
  • Backend & Database: Supabase (PostgreSQL + Auth + Storage)
  • Hosting: Vercel
  • LINE Integration: LINE Messaging API
  • PDF Export: PDFKit + pdfkit-table (ฟอนต์ THSarabunNew)
  • Skeleton Loading: ทุกหน้ามี loading state แบบ skeleton (ยกเว้นหน้าจดงาน)
